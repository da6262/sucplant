## v3.2.15 — 2026-04-15
### 수정
- **주문 SMS 실제 발송 수정**: `sendSMS()` → `sendSolapiSMS()` 직접 호출로 변경. 기존에는 `window.solapi` SDK를 찾다가 브라우저에 없어서 콘솔 로그만 찍히고 실제 발송 안 되던 버그 수정
- **SMS API 인증정보 환경설정 연동**: 환경설정 SMS탭에 솔라피 API Key / API Secret / 발신번호 입력 필드 추가. 저장 후 주문관리·고객관리 모든 문자 발송에 즉시 반영
- **런타임 설정 우선**: `getSolapiConfig()` 함수 추가 — 환경설정 값 우선, 없으면 코드 기본값 폴백
