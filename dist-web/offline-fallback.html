관리시스템 - 오프라인 모드

📡 현재 상태: 오프라인 모드

인터넷 연결이 불안정하거나 서버에 접속할 수 없습니다. 로컬 모드로 작동합니다.

🔧 사용 가능한 기능

  • 주문 관리 (로컬 저장)
  • 고객 정보 관리
  • 재고 관리
  • QR 코드 생성
  • 데이터 내보내기 (Excel)
  • 대시보드 통계

💡 해결 방법

  1. 인터넷 연결 확인: Wi-Fi 또는 모바일 데이터 연결 상태를 확인하세요.
  2. 방화벽 설정: 방화벽이 프로그램을 차단하고 있는지 확인하세요.
  3. 브라우저 새로고침: F5 키를 눌러 페이지를 새로고침하세요.
  4. 캐시 삭제: 브라우저 캐시를 삭제하고 다시 시도하세요.

📞 기술 지원

문제가 계속 발생하면 다음 정보를 확인해주세요:

  • 브라우저: Chrome, Firefox, Safari, Edge 등
  • 운영체제: Windows, Mac, Android, iOS
  • 오류 메시지: 개발자 도구(F12)의 콘솔 탭 확인
