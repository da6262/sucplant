데이터 저장 방식

로컬 모드 (현재 활성)

• 저장 위치: 브라우저 LocalStorage
• 장점: 빠른 속도, 오프라인 사용 가능
• 단점: 브라우저별로 데이터 분리
• 백업: 수동으로 JSON 파일 다운로드

API 모드 (서버 연결 시)

• 저장 위치: 서버 데이터베이스
• 장점: 여러 기기에서 동기화
• 단점: 인터넷 연결 필요
• 백업: 서버에서 자동 백업
