🌐 브라우저별 저장소 테스트

⚠️ 중요한 정보

LocalStorage는 브라우저별로 독립적입니다. Chrome에서 저장한 데이터는 Firefox에서 볼 수 없고, Firefox에서 저장한 데이터는 Edge에서 볼 수 없습니다.

🖥️ 현재 브라우저 정보

💾 LocalStorage 상태

🧪 테스트 데이터 생성

📋 브라우저별 테스트 가이드

Chrome: 이 페이지에서 테스트 데이터를 생성하고, Chrome 개발자 도구(F12) → Application → Local Storage에서 확인
Firefox: Firefox에서 같은 페이지를 열고, Firefox 개발자 도구(F12) → Storage → Local Storage에서 확인
Edge: Edge에서 같은 페이지를 열고, Edge 개발자 도구(F12) → Application → Local Storage에서 확인
Safari: Safari에서 같은 페이지를 열고, Safari 개발자 도구 → Storage → Local Storage에서 확인

🔄 데이터 동기화 방법

1. 데이터 내보내기/가져오기: 현재 브라우저에서 데이터를 JSON 파일로 내보내고, 다른 브라우저에서 가져오기
2. 서버 API 사용: 모든 브라우저가 같은 서버 API에 연결하여 데이터 동기화
3. 클라우드 동기화: Google Drive, Dropbox 등을 통한 데이터 동기화
